Documentação

A biblioteca de ícones da Huta Dev foi criada para oferecer uma forma simples, rápida e directa de utilizar símbolos em projectos web. Todos os ícones são apresentados visualmente, permitindo ao utilizador copiar o respectivo código e aplicá-lo no seu projecto. Esta documentação explica como tudo funciona e como tirar o melhor proveito da biblioteca.

1. Como os ícones funcionam

Cada ícone é representado através de uma entidade numérica HTML . Uma entidade numérica é um código especial que, ao ser colocado no HTML, é automaticamente transformado no símbolo correspondente. Esta abordagem não exige ficheiros externos como CDN ou script, não utiliza SVG e não acrescenta peso ao carregamento da página.

Exemplo de entidade numérica:

&#9728;

Ao usar esse código no HTML, o navegador renderiza o símbolo ☀. Este método garante desempenho elevado, compatibilidade com praticamente todos os navegadores e funcionamento estável independentemente do ambiente.

2. Como copiar o ícone

O processo de utilização é directo. O utilizador navega pela página dos ícones , escolhe o ícone clicando nele, 0 modal apresenta o símbolo e o seu código, pronto a ser cópiado e colado no HTML do projecto.

Exemplo de uso no HTML:

<button> &#9728; </button>

Resultado:

Ao colar este código, o símbolo aparecerá imediatamente na página, sem a necessidade de qualquer recurso adicional.

3. Vantagens de utilizar entidades numéricas

A biblioteca foi construída com entidades numéricas por várias razões práticas:

  • Carregamento extremamente rápido;
  • Compatibilidade universal com HTML;
  • Dispensa de ficheiros SVG ou dependências externas;
  • Integração simples com qualquer estilo ou componente;
  • Ideal para projectos leves, rápidos e optimizados.
